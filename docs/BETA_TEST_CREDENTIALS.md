### Get User JWT Tokens (for testing)

```sql
SELECT 
  st.sessionId,
  st.usuarioId,
  u.nome,
  st.refreshToken,
  st.expiresAt
FROM "SessaoToken" st
JOIN "Usuario" u ON st.usuarioId = u.usuarioId
WHERE u.email LIKE '%teste.imobi.com'
  AND st.revogadoEm IS NULL
ORDER BY st.criadoEm DESC;
```

### View Test Payments (Stripe)

```sql
SELECT 
  pagamentoId,
  usuarioId,
  valor,
  status,
  stripePaymentId,
  stripeStatus,
  criadoEm
FROM "pagamentos"
WHERE usuarioId IN (
  SELECT usuarioId FROM "Usuario"

)
ORDER BY criadoEm DESC;
```
