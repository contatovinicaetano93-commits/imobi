                        KEY FEATURES SUMMARY
================================================================================

TYPE CHECKING:
  - Runs on every push and pull request
  - Validates TypeScript across all packages
  - Fails if any type errors detected
  - Duration: ~2-3 minutes

BUILDING:
  - Runs on every push and pull request
  - Validates entire monorepo build
  - Uploads artifacts for 5 days
  - Uses pnpm cache for speed
  - Duration: ~5-10 minutes

SECURITY:
  - Runs weekly on Thursday
  - Scans all dependencies for vulnerabilities
  - Auto-creates GitHub issues if found
  - Can be triggered manually
  - Duration: ~2 minutes

DEPENDABOT:
  - Auto-updates npm dependencies weekly
  - Auto-merges patch + minor versions
  - Creates PRs for major versions
  - Auto-updates Docker images
  - Auto-updates GitHub Actions
  - Limit: 5 PRs (npm), 3 PRs (docker/actions)

PRE-PUSH:
  - Runs locally before push
  - Prevents TypeScript errors from being committed
  - Each developer must install once
  - Command: ./scripts/setup-hooks.sh
