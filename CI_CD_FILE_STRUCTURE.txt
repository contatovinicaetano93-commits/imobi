================================================================================
                      WHAT HAPPENS AFTER PUSH
================================================================================

IMMEDIATELY:
  1. GitHub detects new workflow files (5 min wait)
  2. Type-check workflow runs
  3. Build workflow runs (parallel to type-check)
  4. Both workflows should complete in ~10-15 minutes

WITHIN 24 HOURS:
  1. Dependabot runs and creates PRs if updates available
  2. Auto-merge kicks in for patch/minor updates
  3. GitHub Actions workflows show up in repository Actions tab

WEEKLY:
  1. Monday: Dependabot runs (3 update checks, 1 hour apart)
  2. Thursday: Security audit runs and creates issues if vulnerabilities
