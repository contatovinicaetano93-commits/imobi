## **CONTEXTO ESTRATÉGICO**

PME construtoras precisam de capital para executar obras, mas:

- ❌ Juros altos (2-3% a.m.)
- ❌ Risco concentrado em uma empresa
- ❌ Sem acompanhamento técnico

### **Solução imbobi:**
✅ Validação tripla (GPS + eng. presencial + parceiros)  
✅ Taxa competitiva (0.99% a.m. ou menos)  
✅ Aprovação rápida (1-7 dias vs 30-60 dias)  
✅ Acompanhamento técnico (reduz risco)  
✅ **Pulverização de risco via fundos**

## **PULVERIZAÇÃO DE RISCO**

### **Modelo Atual (Banco Tradicional):**
```
Banco investe R$100M em 1 grande cliente
├─ Risco: Concentrado
├─ Default: Perde R$100M
└─ ROI: Variável (depende de 1 cliente)
```

### **Modelo imbobi (Pulverizado):**
```
Fundos investem R$100M via imbobi em 50-100 obras
├─ Obra 1: R$2M (default = perde R$2M)
├─ Obra 2: R$3M (default = perde R$3M)
├─ Obra 3: R$2.5M (default = perde R$2.5M)
│ ...
├─ Obra 100: R$2M (default = perde R$2M)
│
├─ Risco: Distribuído
├─ Default esperado: 3-5% (R$3-5M em R$100M)
├─ Outcome: Lucro mesmo com alguns defaults
└─ ROI: Previsível (~9-12% a.a. após inadimplência)
```

### **Vantagem do Modelo:**
- **Curva de Gauss:** Alguns clientes vão dar default, mas maioria paga
- **Estatística:** Com 100 obras de R$2M, default de 1-2 não quebra o retorno
- **Escala:** Quanto mais obras, mais seguro

## **VALIDAÇÃO TRIPLA: CRÍTICA PARA FUNDOS**

Porque fundos investem bilhões, validação é **incontornável**:

### **Camada 1: GPS Digital**
- ✅ Geo-validação automática
- ✅ Comprovação de obra real
- ✅ 0% custo adicional
- ✅ Velocidade: <1 hora

### **Camada 2: Engenheiro Presencial**
- ✅ Vistoria técnica
- ✅ Avaliação de fundação, alvenaria, cronograma
- ✅ Relatório estruturado
- ✅ Custo: R$1k-2k por visita
- ✅ Velocidade: 2-5 dias

### **Camada 3: Parceiros Regionais**
- ✅ Cobertura nacional
- ✅ Urgências 24h
- ✅ Validação de crédito local
- ✅ Custo: R$500-800 por visita

### **Mitigação de Risco:**
```
Problema             Mitigação
─────────────────────────────────────
Obra para, crédito vira tóxico
→ Eng. presencial identifica atraso
→ imbobi monitora + libera parcela conforme progresso
→ Risco reduzido de 10% para 3-4%

Construtor foge com dinheiro
→ Validação tripla prova obra real
→ Desembolso progressivo (por etapa, não tudo)
→ Eng. acompanha execução
→ Default quase impossível

Taxa inadimplência sobe
→ Scoring dinâmico (score melhora/piora)
→ Acompanhamento técnico (feedback reduz risco)
→ Portfolio diversificado (1 default não quebra)
```

---

## **ACOMPANHAMENTO TÉCNICO = REDUÇÃO DE RISCO**

Para fundos que investem R$100M+, acompanhamento técnico não é nice-to-have, é **obrigatório**:

### **O que imbobi fornece:**
- **Relatórios bi-semanais:**
  - Progresso da obra (cronograma vs realizado)
  - Identificação de riscos (atrasos, qualidade)
  - Recomendações de ação

- **Scoring Dinâmico:**
  - Score do cliente melhora/piora
  - Próximo empréstimo reflete histórico
  - Incentiva performance

- **Liquidação Segura:**
  - Desembolso por etapa (não tudo de uma vez)
  - Cada etapa = visita de eng.
  - Reduz risco de desvio de capital
