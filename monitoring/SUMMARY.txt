🚨 CRITICAL ALERTS (P0 + P1)
════════════════════════════════════════════════════════════════════════════

P0 - CRITICAL (Auto-escalate PagerDuty + Slack + Email)
├─ DB Connection Pool Exhausted     → Queued > 10 for 1 min
├─ Database Connection Failures     → Any connection error
├─ Disk Space Critical              → <10% available
├─ Dead Letter Queue Has Messages   → >0 unprocessable jobs
└─ Uptime SLA Breach               → Error rate >5% for 1h

P1 - HIGH (Slack + Email)
├─ HTTP Error Rate > 5%             → 5% errors for 5 min
├─ Latency P95 > 500ms              → Degraded performance
├─ S3 Upload Failures               → >3 failures in 5 min
├─ Auth Failures Spike              → >20 failures in 5 min
└─ Slow Database Queries            → >5 slow queries in 5 min
