TROUBLESHOOTING
─────────────────────────────────────────────────────────────────────────────

If rate limiting not working:
→ Check auth.controller.ts for @Throttle decorator
→ Expected: @Throttle({ default: { limit: 10, ttl: 60000 } })

If IDOR test failing (user 2 sees user 1's data):
→ Check services for ownership validation
→ Expected pattern: if (resource.usuarioId !== userId) throw new ForbiddenException()

If plaintext password in response:
→ Check auth service response DTO
→ Should exclude 'senha' and 'password' fields

If authorization test failing:
→ Check controllers for @UseGuards(JwtAuthGuard)
→ All protected routes must have this guard

See SECURITY_TEST_SUMMARY.md for detailed troubleshooting.
